# STM32_wificar
A smart car using ceramic substrates, STM32F103ZET6 board and ESP8266 WiFi module

Hardware
------------------
* STM32F103ZET6 board
* L293DD
* 78M05
* 78M12
* ESP8266(E103-W01/E103-W01-IPX)
* DC motors

Requirements
__________________
**Building**
* MDK Version 4+

**Debuggers**
* ST-LINK/V2
* USB to TTL(CH340G)
